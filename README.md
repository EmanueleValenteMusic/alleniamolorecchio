# Alleniamo l'orecchio

Scegli la modalità che preferisci per allenare il tuo orecchio musicale.

## Uso immediato

Dopo la build, apri `index.html` con un doppio click. Non serve alcun server.

## Sviluppo

```bash
npm install
npm run build
```

Il comando di build genera un unico file HTML autosufficiente sia in `index.html` sia in `dist/index.html`.
